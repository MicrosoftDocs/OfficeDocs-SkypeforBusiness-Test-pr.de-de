# Clientvergleichstabellen für Lync Server 2013

 

_**Letztes Änderungsdatum des Themas:** 2017-05-03_

In den folgenden Tabellen werden die Features und Fähigkeiten von Lync 2013-Clients in den folgenden Kategorien verglichen:

  - Unterstützung für die erweiterte Anwesenheit

  - Unterstützung von Kontakten und Kontaktgruppen

  - Chatunterstützung

  - Konferenzunterstützung

  - Telefonieunterstützung

  - Unterstützung externer Benutzer

  - Unterstützung für Archivierung, Richtlinientreue und Protokollierung

In diesen Tabellen werden die Features angegeben, die Lync-Benutzern in einer lokalen Bereitstellung von Lync Server 2013 zur Verfügung stehen. Dieselben Features sind auch, sofern nicht anders vermerkt, für Lync Online- und Office 365-Benutzer verfügbar.


> [!IMPORTANT]
> Unter <A href="lync-server-2013-mobile-client-comparison-tables.md">Vergleichstabellen für mobile Clients für Lync Server 2013</A> finden Sie Details zur Funktionsverfügbarkeit in mobilen Lync 2013-Clients.




> [!NOTE]
> <UL>
> <LI>
> <P>Ausführliche Informationen zu Lync Online-Abonnementplänen finden Sie unter "Lync Online-Dienstbeschreibung" unter <A href="http://go.microsoft.com/fwlink/p/?linkid=282430">http://go.microsoft.com/fwlink/p/?LinkId=282430</A>.</P>
> <LI>
> <P>Lync Server 2013 unterstützt die folgenden früher freigegebenen Clients: Lync 2010, Lync 2010 Mobile, Lync Phone Edition und Lync 2010 Attendant. Informationen zu diesen Clients finden Sie in den Clientvergleichstabellen zur Lync 2010-Version unter <A href="http://go.microsoft.com/fwlink/p/?linkid=213798">http://go.microsoft.com/fwlink/p/?LinkID=213798</A>.</P>
> <LI>
> <P>Der Lync 2013-Client unterstützt Audio und Video in einer VDI-Umgebung (Virtual Desktop Infrastructure). Ausführliche Informationen zu den Lync 2013-Funktionen in einer VDI-Umgebung finden Sie unter <A href="lync-server-2013-supported-virtualization-technologies-and-known-limitations.md">Unterstützte Virtualisierungstechnologien und bekannte Einschränkungen in Lync Server 2013</A>.</P></LI></UL>
